About the ChatGPT Micro-Cap Experiment

This project showcases an AI-managed micro-cap trading portfolio. It provides tools for tracking holdings, logging trades, and visualizing performance.

Using the Tools

  • Process Portfolio: Updates positions with current market prices and recalculates equity.
  • Trade Form: Submit buys or sells by entering ticker, price, shares, and optional stop loss or reason.
  • Trade Log: Review a history of all recorded trades for transparency and analysis.
  • Graphs: View equity history versus benchmarks to measure performance over time.

For more background and research notes, see the GitHub repository.
